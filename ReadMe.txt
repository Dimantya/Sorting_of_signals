Это программа, позволяющая отсортировать заданную комбинацию из сигналов Green, Blue, Red.
Сортировка сигналов происходит из условия, что Greean<Blue<Red. Итоговая кобинация будет отсортирована согласно этой логике.
Для большей наглядности программа имеет графический интерфейс, созданный с помощью библиотеки tkinter для Python.
Программа написана с использованием IDE PyCharm


Для установки библиотеки можно воспользоваться следующей командой:
pip install -r requirements.txt
Или просто ввести в терминале pip install tk


Файл Automiq_main - основной файл, который реализует логику программы.

В файл Tests реализованы тесты, проверяющие правильность списков и их элементов, на основе встроенной библиотеки unittest

Помимо этого в программе реализован функционал логирования ключевых событий в логике программы.
Все логи пишутся в текстовый файл python_logging с указанием времени каждого события. Функционал реализован на основе встроенной библиотеки logging.
В рамках логов также реализованы проверки количества добавленных элементов на экране и их цветов.
